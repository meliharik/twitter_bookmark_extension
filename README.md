# Twitter Bookmark Extension

A Chrome Extension to scrape and save Twitter bookmarks to the organizer app.

## Installation
1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked**.
4. Select this folder (`extension`).

## Usage
1. Click the extension icon in the Chrome toolbar.
2. Click **"Login with Web App"** to authenticate with the running frontend application.
3. Navigate to your Twitter Bookmarks page.
4. Click **"Start Scanning"** to sync your bookmarks.
